# ShoppingSite (Decorem)
Concept for our retro decor and apparel ecommerce site. 

## Dependencies
### Front-end:
axios
react
redux
redux-thunk
### Back-end:
bcrypt
cookie-parser
cors
dotenv
express
jsonwebtoken
knex
nodemon
pg (postgres)


RESTful API for Shopper & Creator side of the app. item CRUD on Creator side and view and "purchase" of items on Shopper side.
